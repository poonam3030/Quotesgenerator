 Positive Affirmation Quote Generator
This project is a simple web application that generates positive affirmation quotes using a public API. The quote automatically refreshes every 1 minute, so users keep getting new motivational messages without needing to click anything.

What This Project Does
The website displays uplifting affirmation quotes to help create a positive mindset.  
Once the page loads:
- A quote appears on the screen
- Every 60 seconds, a new quote is fetched automatically
- The update happens smoothly without reloading the page

 Technologies Used
 1. HTML – Structure of the webpage  
 2. CSS – Styling and layout design
 3. JavaScript – Fetching quotes and handling automatic refresh
 4. Public Quotes API – Source of affirmation quotes  

Learning Outcomes
1.How to fetch data from an API using JavaScript  
2.How to update webpage content dynamically  
3.Using timers (`setInterval`) for automatic updates  
4.Improving UI simplicity and readability  

This is a beginner-friendly front-end project focused on learning API integration and dynamic web content.
